dart

inRangeOf(min, max) ​

Clamps the number within the given range.

  • min: Minimum allowed value.
  • max: Maximum allowed value.

Throws Exception if either min or max is null. Throws ArgumentError if min is greater than max.

dart
var num = 5;
print(num.inRangeOf(1, 10)); // Output: 5
var num = 5;
print(num.inRangeOf(1, 10)); // Output: 5
